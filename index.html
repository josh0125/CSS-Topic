CSS Topic - Everything to Do with Text

Text is an important part of any website, and many times we want our texts to look different or have different attributes that it does, like being aligned to the right or underlined. This website will help you understand how to code the CSS for text-alignments, text-decorations, text-transformations, text-margins, and line heights.
